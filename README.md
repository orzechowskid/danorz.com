# Prerequisites

- install node@^14.0.0
- install npm@^7.0.0
- install Docker

# Getting Started

- clone this repo
- bootstrap the repo: `npm run bootstrap`
- build some Docker images and run them in containers: `./up.sh`
- start the frontend: `cd packages/frontend; npm run start`
- (in your browser) visit http://localhost:8080
